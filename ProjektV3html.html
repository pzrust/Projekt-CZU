Úvod

Jako téma mého projektu jsem si zvolil Business Network. Jedná se o pomyslnou sociální sít, kde se setkávají lidi z byznysu a mohou si navzájem posílat zprávy atd. Jako inspirace zde posloužila sociální síť LinkedIN. Tato dokumentace popisuje metodiku, která byla při projektu použita.

Metodika

Proces tvoření tohoto projektu začal návrhem modelu pro nastaveí aplikace "Business Network". Tento model byt sestaven a napsán v XML, ve kterém bylo použito mnoho elementů, atributů a entit. Následně byl vytvořen soubor XSD, což je XML Schema, do kterého se následně doplní záznamy. Po tomto se vytvořil XSLT soubor, který je nutný pro transformaci do JSON. Na závěr byla vytvořena tato dokumentace ve značkovacím jazyce HTML.

Transformace

Ze mnou vytvořeného XML souboru jsem vytvořil další soubor XSL, který je nutný k transformaci. Úspěšně mi proběhla transformace z XSL do JSON, což je JavaScript Object Notation. JSON se používá pro výměnu ale i pro ukládání dat. Funguje mezi mobilní aplikací a serverem.

Závěr

Práce se celkově povedla, všechny části projektu úspěně proběhly validátory. Práce na tomto projektu mi dala mnoho nových poznatků, naučil jsem se základy XML, XSD, XSLT a HTML. Tato práce pro mě byla přinosem.

Při zpracování tohoto projektu jsem často využíval tento web jako zdroj informaci. Veěkeré soubory spjaty s tímto projektem jsou ulozeny zde na GitHubu.
